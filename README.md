# Geometry Dash Extension for MakeCode Arcade

**Adds GD-style tiles, gamemodes, physics, portals, spikes, and orbs.**

## Features
- 7 gamemodes with different controls/physics
- Tile helpers for tilemaps (ground, deadly spikes, portals, gravity flip)
- Auto-jump, gravity flip, ship flying, etc.
- Easy level setup

See `main.ts` for full API.
